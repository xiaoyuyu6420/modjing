## 三、模镜独特优势（差异化）

### 1. 企业合规专区 `/enterprise`（hvoy 没有）

| 字段 | 说明 |
|------|------|
| 发票支持 | 增值税普票/专票/电子发票 |
| 等保级别 | 等保二级/三级、ISO27001 |
| 数据位置 | 国内/国外/混合 |
| SLA 承诺 | 在线率承诺、故障响应时间 |
| 企业账户 | 是否支持企业账户/子账号 |
| 合同模板 | 是否提供合同模板 |

### 2. 权重公开 + 可调（hvoy 黑箱）

- `/methodology` 公开 32 道探针 prompt
- 公开 4 维指纹算法
- **权重滑块**：用户可拖动调整 W1-W5 看排名变化
- 任何一次评分都可复现

### 3. RP 垂直深耕（hvoy 有但没深耕）

| 字段 | 说明 |
|------|------|
| 出戏率 | 模型"出戏"（回复不像角色）的比例 |
| 审查程度 | 内容审查严格程度 |
| 上下文窗口 | 支持的最大上下文 |
| "活人感"评分 | 回复是否有人味 |

### 4. Coding Plan 独立品类（hvoy 混在一起）

- `/plans` 把 GPT Plus / Claude Max / Codex 订阅作为独立品类
- 对比：价格、模型覆盖、并发限制、重置规则

### 5. 双探针对比（hvoy 单一来源）

- 同时展示 hvoy 探针结果 + 模镜自建探针结果
- 用户可对比两家数据，增加可信度
